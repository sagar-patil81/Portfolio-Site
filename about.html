About Me

I have been working with core web technologies like HTML, CSS, and JavaScript to create visually appealing and responsive interfaces. Along with frontend development, I’m also exploring backend technologies such as Node.js and Express, where I’ve learned to handle form submissions, build APIs, and integrate email services into web applications.

One of my key strengths is my willingness to learn and experiment. I actively try to improve my skills by building real-world projects like portfolio websites and contact form systems that connect frontend interfaces with backend functionality.

I focus on writing structured, readable code and aim to create smooth user experiences. I am particularly interested in full-stack development and continuously working towards expanding my knowledge in both frontend and backend technologies.

Currently, I am looking for opportunities to apply my skills, gain real-world experience, and grow as a developer.
